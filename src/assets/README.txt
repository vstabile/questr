PIXEL ART ASSETS FOR QUESTR

This directory should contain the following pixel art assets:

1. iso-tileset.png
   - Size: 256x256 pixels
   - Contains isometric tiles for the game world
   - Each tile should be 64x32 pixels (width x height)
   - First tile (index 0) should be a grass/ground tile

2. player.png
   - Size: 128x160 pixels (4 columns x 5 rows of frames)
   - Each frame should be 32x32 pixels
   - Contains the following animation frames:
     - Row 1 (frames 0-3): Idle animation
     - Row 2 (frames 4-7): Walking down
     - Row 3 (frames 8-11): Walking left
     - Row 4 (frames 12-15): Walking right
     - Row 5 (frames 16-19): Walking up

You can create these assets using pixel art tools like:
- Aseprite
- Piskel
- GIMP
- Photoshop with pixel art settings

Place the completed assets in the images/ directory.
